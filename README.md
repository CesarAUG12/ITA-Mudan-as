# ITA Mudanças - Website Institucional

Projeto de website institucional desenvolvido para a empresa ITA Mudanças, focado em apresentar os serviços da empresa, facilitar o contato com clientes e oferecer uma experiência responsiva e moderna.

## 🚀 Funcionalidades

* Layout totalmente responsivo
* Integração com WhatsApp para solicitações de orçamento
* Formulário personalizado para coleta de informações
* Seções de serviços, diferenciais e regiões atendidas
* Navegação fluida entre seções
* Interface otimizada para desktop e dispositivos móveis
* Organização visual voltada para experiência do usuário (UI/UX)

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Lucide Icons

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados conceitos importantes como:

* Responsividade
* Estruturação semântica HTML
* Organização de layout com Flexbox e Grid
* Integração de formulário com WhatsApp
* Ajustes visuais e refinamentos de UI
* Registro e configuração de domínio
* Publicação de site

Além disso, o projeto também contou com apoio do Cursor AI para acelerar parte do desenvolvimento e auxiliar na resolução de problemas durante o processo.

## 🎯 Objetivo

Criar um site moderno, simples e funcional para melhorar a presença digital da empresa e facilitar o contato de novos clientes.
